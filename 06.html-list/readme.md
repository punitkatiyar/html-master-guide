## Description List

> dl : defines the description
> dt : defines the term (name)
> dd :  describes each term in a description list


```
<dl>
  <dt>html</dt>
  <dd>- Create Document</dd>
  <dt>CSS</dt>
  <dd>- Style the Document</dd>
</dl>
```
